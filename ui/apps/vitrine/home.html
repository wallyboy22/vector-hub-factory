Bem-vindo à Fábrica.

O Vector Hub Factory não é apenas uma interface de chat. É uma linha de montagem técnica para transformar Documentos Científicos Complexos em oráculos consultáveis via IA.

Nós recebemos o PDF, fragmentamos o conhecimento em pequenos Chunks semânticos, calculamos coordenadas multidimensionais (Embeddings) e o servimos através de uma cadeia Multi-LLM resiliente.

🧠 RAG Dinâmico 🌐 Multi-Provedor (Groq/Gemini) ⚡ Fallback Automático
